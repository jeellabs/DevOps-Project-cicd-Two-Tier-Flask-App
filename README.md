# Two-Tier Flask App — DevOps Project

## Tech Stack
- AWS EC2, Docker, Docker Compose, Jenkins, MySQL, Python Flask

## Architecture
Flask App + MySQL — both containerized via Docker Compose
CI/CD via Jenkins + GitHub Webhooks

## How to Run
docker compose up -d --build
